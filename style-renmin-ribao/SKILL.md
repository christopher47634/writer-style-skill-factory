# 人民日报写作风格

## 成稿优先协议

- 用户提供主题和材料后，默认直接交付完整稿件；未要求时不输出分析、规则、提纲或自检。
- 内部先构思 3 个中心论点表述和 2 种论证次序，选择政策依据最稳、层次最清楚的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖立论、总分结构、论据、排比节奏、四字格控制和建设性结尾。
- 初稿后静默修订两轮：先检查论点与证据对应，再删除通用 AI 腔、空洞口号、无来源政策判断和过密排比。
- 默认服从用户指定的长度和文体；未给长度时生成约 1000 字评论。
- 不冒充该媒体，不生成虚假署名、栏目、政策引语或权威结论。
- 材料不足时降低断言强度并标记待核信息，不补造政策、数字和讲话。
- 默认输出只有标题与正文。

> 分析基础：21篇评论类文章（2026年6月），主要涵盖人民锐评、壹时评、思想纵横、中国道路中国梦等栏目。
> 样本边界：不同栏目有独立编辑传统；环球时报社评等外部品牌不能作为人民日报核心文风证据。生成时必须按具体栏目和用途选择规则。

## 一句话定义
政治挂帅的论证式写作——先引权威定调，再层层论证，最后升华号召。雅俗兼备，修辞程式化。

---

## 一、选题层

| 维度 | 特征 |
|------|------|
| 题材类型 | 政策解读、时事评论、社会现象点评、国际博弈、思想建设 |
| 切入角度 | 从政策/领导人讲话/重大新闻事件切入，不做小场景描写 |
| 问题意识 | 「这项政策意味着什么？」「我们应该怎么看/做？」 |
| 冲突来源 | 认识偏差、执行偏差、外部挑战、发展矛盾 |
| 关注对象 | 国家、社会、行业、群众（作为集体概念，非个体） |
| 选题气质 | 正面、建设性、引导性。批评也是建设性批评 |

选题服务于「统一思想、凝聚共识」。不是记录事件，而是解释意义。

---

## 二、立场层

| 维度 | 特征 |
|------|------|
| 作者位置 | 解释者/引导者。代表「正确立场」说话 |
| 对人物态度 | 集体视角，不关注个体命运 |
| 对社会态度 | 建设性。问题存在但可以解决，方向是对的 |
| 判断方式 | 直接下结论，而且是权威结论 |
| 价值底色 | 乐观、坚定、自信 |

核心规则：**立场必须鲜明，不能含糊。判断必须来自权威来源（领导人讲话/政策文件/中央精神）。**

---

## 三、结构层

### 开头（4种范式）

**1. 领导人讲话引用（45%）**
```
习近平总书记强调："安全是发展的前提，发展是安全的保障。"这一重要论述……
```

**2. 新闻事件引入（30%）**
```
近日，三大电信运营商推出面向个人、企业的"词元（Token）套餐"……
```

**3. 设问/悬念（15%）**
```
理解今天的中国，需要一些注脚。
```

**4. 群众声音（10%）**
```
"八个手机组成地动仪""条条大路通电商"，网友的段子既无奈又好笑……
```

### 中段推进：总—分—总

典型推进链：
```
权威定调 → 分论点1（例证+分析）→ 分论点2 → 分论点3 → 总结升华
```

分论点常用：「从……看」「一方面……另一方面」「不仅……而且……更……」

### 结尾（4种）

1. **展望号召（50%）**：「向着更远大的目标进发，需要心中有火、眼中有光。」
2. **总结升华（25%）**
3. **首尾呼应（15%）**
4. **署名+日期**

---

## 四、叙事层

| 维度 | 特征 |
|------|------|
| 叙事视角 | 全知视角/权威视角。作者代表正确立场 |
| 镜头距离 | 以远景（宏观/政策/时代）为主，偶尔中景（行业/地区） |
| 时间处理 | 以现在和未来为主，历史作为论据 |
| 场景密度 | 低。几乎不写场景，用案例代替场景 |
| 人物呈现 | 人物作为案例出现，不塑造个体 |
| 信息释放 | 直接给结论，然后论证 |
| 叙事速度 | 快。不铺陈，直奔论点 |

**论证感**：不像纪录片，像社论/演讲稿。

---

## 五、段落层

典型段落功能分布：
- 判断段：35%（给出观点/定调）
- 信息段：25%（提供案例/数据）
- 论证段：25%（分析推理）
- 号召段：15%（展望/要求）

段落较长，逻辑严密。短段极少——每段都在推进论证。

---

## 六、句子层

| 指标 | 特征 |
|------|------|
| 句子长度 | 长句为主，50-80字常见 |
| 停顿方式 | 逗号密集，句号收束 |
| 判断句 | 高频。「是……的」「意味着」「体现了」 |
| 转折句 | 「但」用于纠偏，「然而」用于承认问题 |
| 排比句 | 几乎每篇必用。「从……到……再到……」 |
| 疑问句 | 设问为主，自问自答 |
| 口语程度 | 书面为主，偶有网络语言 |

四字格密集：「越险越进、越陡越攀」「蹄疾步稳、行稳致远」

---

## 七、词语层

### 推荐词库（人民日报专用）
```
征程、奋进、笃行、擘画、锚定、赋能、高质量发展、新发展格局、
人民至上、系统观念、守正创新、底线思维、蹄疾步稳
```

### 禁用词库
```
完蛋、崩溃、药丸、躺平（除非在批评语境中引用）、
阴阳怪气、负能量、历史虚无主义
```

### 引语层级
1. 习近平讲话（最高权威，几乎每篇必引）
2. 古语典故（增强文化厚度）
3. 网友言论/群众语言（接地气）
4. 案例当事人（具体化）

---

## 八、情绪层

| 维度 | 特征 |
|------|------|
| 情绪强度 | 5-6/10。积极向上但不夸张 |
| 情绪外露 | 中等。有情感表达但受控 |
| 情绪来源 | 来自信念和方向感，不来自个体故事 |
| 情绪曲线 | 开头定调 → 中段论证（理性为主）→ 结尾升华（情感上升） |
| 是否克制 | 批评时克制，鼓励时可适度热烈 |
| 是否反讽 | 极少。正面论述为主 |
| 底色 | 乐观、坚定、自信 |

---

## 九、传播层

| 维度 | 特征 |
|------|------|
| 标题类型 | 栏目标签+主题、对仗/比喻、动宾结构 |
| 开头吸引力 | 靠权威性和时效性抓人 |
| 信息钩子 | 政策解读、热点回应 |
| 金句密度 | 高。排比句和四字格是传播载体 |
| 转发理由 | 代表官方态度、提供解读框架 |
| 平台适配 | 报纸+公众号同步 |

### 栏目风格差异

| 栏目 | 风格 | 篇幅 |
|------|------|------|
| 人民锐评 | 平实有力 | 800-1200字 |
| 壹时评 | 文学性强 | 600-1100字 |
| 思想纵横 | 理论性强 | 1000-4000字 |
| 中国道路中国梦 | 温情叙事 | 1300-1400字 |

## 写作自查清单

- [ ] 选题是否服务于「统一思想/凝聚共识」？
- [ ] 开头是否引用权威来源定调？
- [ ] 结构是否总—分—总？
- [ ] 分论点是否有层次（不仅/而且/更）？
- [ ] 是否有排比句增强气势？
- [ ] 四字格是否适度？
- [ ] 引语是否来自权威来源？
- [ ] 结尾是否有展望号召？
- [ ] 情绪基调是否积极向上？
- [ ] 批评是否以建设性意见收尾？

---

## 执行协议

1. 先确认用途：政策解读、评论、理论文章还是群众叙事。栏目不明时采用清楚、克制的评论体，不强行套“权威定调”。
2. 领导人讲话、政策表述、文件名称和数字必须来自用户材料或可靠来源；不得自行生成“权威引语”。
3. 将事实、政策原文、解释和价值判断分开写，避免用宏大措辞代替论证。
4. 排比、四字格和号召式结尾按需使用；若用户要普通评论或内部说明，应降低程式化表达。
5. 不冒充人民日报，不虚构栏目名、署名、发布日期或官方立场。

**优先级：政策与引文准确 > 论证完整 > 栏目适配 > 修辞气势。**
