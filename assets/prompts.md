## Prompt 1 — 项目 Logo / Icon（512×512）

**用途：** GitHub 仓库图标、社交媒体头像、favicon
**尺寸：** 512×512 正方形

### 中文版

请生成一张 512×512 的正方形图标设计。

【主题】"Writer Style Skill Factory"——一个将伟大作家写作技艺蒸馏为 AI Skill 的开源项目的 Logo。

【核心视觉】画面中央是一个高度抽象的符号，融合了两个意象：
- 上半部分是一支古典鹅毛笔的笔尖，笔尖向下，线条锐利
- 下半部分是三个相连的电路节点（圆点+连线），呈三角形排列，代表 AI/数字
- 笔尖的墨水滴刚好落在顶端的电路节点上，形成"文学流入技术"的视觉叙事
- 整体形状简洁有力，像一个图腾或徽章

【背景】纯深色 #0d1117，无纹理，让符号悬浮于纯黑中

【色彩】
- 笔尖：蓝紫渐变（#58a6ff → #bc8cff），有微妙的发光效果
- 墨水滴：粉色 #f778ba
- 电路节点和连线：金色 #f0883e → #f7c948 渐变
- 节点圆点内部有微弱的翠绿 #3fb950 光点

【风格】扁平化 + 微拟物，线性图标风格，线条粗细 2-4px，干净利落，无多余装饰

【光影】符号本身有柔和的外发光（glow），颜色与笔尖渐变一致，仿佛在暗处自发光

【要求】
- 缩小到 64×64 仍能辨认出"笔尖+节点"的基本形态
- 不要任何文字、字母、汉字
- 不要写实风格，不要 3D 渲染，保持 2D 扁平

### English Version

Generate a 512×512 square icon design.

Theme: Logo for "Writer Style Skill Factory" — an open-source project distilling great writers' craft into AI skills.

Central visual: A highly abstract symbol fusing two motifs:
- Upper half: a classical quill pen nib pointing downward, with sharp clean lines
- Lower half: three interconnected circuit nodes (circles + lines) arranged in a triangle, representing AI/digital
- An ink drop from the nib lands precisely on the top circuit node, creating a "literature flows into technology" narrative
- Overall shape is minimal and powerful, like a totem or emblem

Background: Pure dark #0d1117, no texture — the symbol floats in deep black.

Colors:
- Quill nib: blue-to-purple gradient (#58a6ff → #bc8cff) with subtle glow
- Ink drop: pink #f778ba
- Circuit nodes and connections: gold gradient #f0883e → #f7c948
- Each node center has a faint emerald #3fb950 light point

Style: Flat design with subtle material feel, linear icon style, stroke weight 2–4px, clean and sharp, no decorative clutter.

Lighting: The symbol itself has a soft outer glow matching the nib gradient, as if self-luminous in darkness.

Requirements:
- Must remain recognizable at 64×64 (ink drop + nodes silhouette)
- No text, letters, or characters of any kind
- No photorealism, no 3D rendering — strictly 2D flat
