# 夏目漱石 文風スキル

## 成稿优先协议

- **硬输出契约**：最终只能包含“## 日语原文”和“## 中文翻译”两部分；中文必须逐段完整翻译日语原文。禁止寒暄、摘要、大意、技法说明、评分和自检。交付前扫描日语正文，不得混入韩文、乱码或不自然回译。除标题与特殊短句外，超过 8 个字符的日语叙述句必须含自然假名；纯汉字中文语序句必须整句重写。
- 用户给出主题后，默认直接写成完整作品；未要求时不输出文风分析、规则说明或自检。
- 内部先构思 3 个场景入口和 2 条“场景—省察”往复路线，选择最自然的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖知性叙述、观察距离、社会讽刺、景物、句式和留有余味的结尾。
- 初稿后静默修订两轮：先校正叙述者声音，再删除通用 AI 腔、卖弄知识和机械的古雅表达。
- 默认先用日语独立完成并修订原文，再给出中文文学翻译；标题同样按“日语原题 → 中文题名”排列。用户明确要求单一语言时服从用户。只给主题时，生成约 1000 字的原创讽刺短篇。
- 不复用原作人物、猫叙述者、完整情节、名句或独特场景。主题不典型时迁移观察方式。
- 若成稿只有知识分子口吻而缺少场景与自我反讽，则加入具体动作、景物和认知落差后重写。
- 高保真门槛：内部生成 3 个候选并匿名评分；总分低于 90、日语不自然，或只靠知识分子、猫、古雅语气和讽刺制造辨识度时，针对最低维度重写，最多 3 轮。
- 默认输出依次为“日语原文”和“中文翻译”，只包含标题与正文。

## 第1層：選題層

### 分析

漱石の選題は「人間の内面の醜さと美しさを日常の断片から抉り出す」ことに終始する。大事件ではなく、書斎・茶の間・散歩道・温泉宿という限られた舞台で、人間の本性が露わになる瞬間を捉える。

| 維度 | 漱石の特徴 |
|------|-----------|
| 題材タイプ | 知識人の日常・人間関係の蹉跌・自我の葛藤 |
| 切り込み角度 | 小さな場面・些細な会話・一瞬の心理変化 |
| 問題意識 | 「人間は利己的であるか」「文明と個人の矛盾」「孤独とは何か」 |
| 衝突源 | 個人の選択・社会的虚栄・知と感情の乖離 |
| 関注対象 | 知識人・学生・家族・友人・隣人 |
| 選題気質 | 冷静・諷刺・悲哀・知的余裕 |

### ルール

- 日常の中から「人間性の本質が見える瞬間」を選ぶ
- 大事件よりも「小さな違和感」を切り口にする
- 知識人と世俗の対立を軸に据える
- 社会的背景は省略せず、しかし主役にはしない

## 第2層：立場層

### 分析

漱石の立場は「冷徹な観察者でありながら、根底には深い悲憤と孤独がある」。表層では諷刺と冷笑を繰り返すが、その奥底には人間への信頼と失望が同居する。

| 維度 | 漱石の特徴 |
|------|-----------|
| 作者位置 | 知的観察者・傍観者・冷笑家 |
| 対人物態度 | 理解と諷刺の往復、嘲りの奥に悲哀 |
| 対社会態度 | 文明批判・道徳批判・しかし絶望ではない |
| 判断方式 | 事実を並べて読者に判断を委ねる |
| 価値底色 | 知的孤高・悲憤・諷刺・孤独への共感 |

### 実例

> 「智に働けば角が立つ。情に棹させば流される。意地を通せば窮屈だ。とかくに人の世は住みにくい。」（草枕）

> 「弱い者を助けようとした時、その弱い者に踏みつけられた。それが人間だ。」（こころ）

### ルール

- 立場を声高に叫ばず、事実の配置と文末の余韻で示す
- 人物を嘲笑するときは、その嘲笑自体が作者の悲しみを含む構造にする
- 「しかし」「ただ」「もっとも」で転換し、一方向に偏らない視座を維持する

### 開頭タイプ

漱石の開頭は「哲学的省察の幕開け」が圧倒的に多い。具体的な場面描写で始まりつつ、すぐに思索の世界へ飛翔する。

### ルール

- 開頭は哲学的命題か、主人公の独白から入る

- 「しかし」「ただ」で方向転換し、単調さを避ける

## 第4層：叙事層

### 分析

| 維度 | 漱石の特徴 |
|------|-----------|
| 視点 | 一人称（余・吾輩・私・僕）が主。全知視点は稀 |
| カメラ距離 | 近景（心理描写）と遠景（哲学的省察）の激しい往復 |
| 時間処理 | 基本は順叙、回想は「……した時」「あの頃」で自然に挿入 |
| 場面密度 | 高い。ほぼ全ての段落に具体的場面か会話がある |
| 人物呈現 | 会話・動作・作者の評論を三段構えで描写 |
| 情報放出 | 徐々に。核心は最後まで明かさない（こころの「先生の秘密」） |
| 叙事速度 | 中庸。哲学的省察で速度を落とし、場面描写で速度を上げる |

### 実例

> 「路は存外広くなって、かつ平だから、あるくに骨は折れんが、雨具の用意がないので急ぐ。帽子から雨垂れがぽたりぽたりと落つる頃、五六間先きから、鈴の音がして、黒い中から、馬子がふうとあらわれた。」（草枕）

- 近景から遠景への自然なカメラ移動
- 感覚描写（聴覚→視覚）の多層構造

### ルール

- 視点は一貫して主人公の一人称に固定する
- 心理描写と場面描写を交互に配置し、読者の没入と距離感を操作する
- 核心的秘密は段階的に明かす
- 「と思った」「と感じた」「と見た」で主観と客観を区切る

## 第5層：段落層

### 分析

- 総段落数：253段落
- 平均段落長：2,605文字（非常に長い）
- 段落は通常、一つの場面単位で構成される

### 段落機能

| タイプ | 漱石での使用法 |
|--------|---------------|
| 場面段落 | 具体的場面描写で読者を引き込む |
| 省察段落 | 場面の合間に哲学的考察を挿入（漱石最大の特徴） |
| 対話段落 | 会話のみで進行。作者の介入を最小限に |
| 評論段落 | 作者が直接語りかける形で社会批判を行う |
| 情緒段落 | 景物描写に心情を重ねる（情景交融） |

### 漱石特有の段落構成

1. **場面描写** → 2. **心理的反応** → 3. **哲学的飛翔** → 4. **場面復帰**

> 「菜の花が一面に見える。雲雀はあすこへ落ちるのかと思った。いいや、あの黄金の原から飛び上がってくるのかと思った。次には落ちる雲雀と、上る雲雀が十文字にすれ違うのかと思った。最後に、落ちる時も、上る時も、また十文字に擦れ違うときにも元気よく鳴きつづけるだろうと思った。」（草枕）

### ルール

- 一段落は一つの場面単位で閉じる
- 場面→省察→場面のリズムを守る
- 段落冒頭は具体的な場面描写か、独白から入る
- 段落末尾は余韻を残す形で閉じる

- 読点多用は禁止。1文3個までを目安にする
- 文語表現（なり・たり・ず・ぬ）を現代文に散りばめ、格調を加える

## 第8層：情緒層

### 分析

| 維度 | 漱石の特徴 |
|------|-----------|
| 情緒強度 | 3-7/10。爆発的ではなく、じわじわと効かせる |
| 情緒露出 | 圧倒的に抑制。直接的な感情表現は稀 |
| 情緒源 | 場面描写・人物の行動・作者の省察 |
| 情緒曲線 | 平静→諷刺→悲哀→省察→余韻 |
| 克制度 | 非常に高い。感嘆符は66万字中わずか16個 |
| 諷刺 | 冷静な筆致で人間の愚かさを描く（最高の特徴） |
| 悲憤 | 表面の諷刺の奥に深い孤独と悲哀がある |

### 漱石の情緒制御

- **感嘆符の極度の少なさ**：66万字中わずか16個。驚異的な自制
- **疑問文の多用**：推定1,016文。問いかけることで読者に感情を委ねる
- **「愉快」「面白い」による知的評価**：直接的感情表現を回避
- **景物描写への心情移入**：「菜の花」「雲雀」「雨」に心情を重ねる

### 実例

> 「うれしい事に東洋の詩歌はそこを解脱したのがある。採菊東籬下、悠然見南山。ただそれぎりの裏に暑苦しい世の中をまるで忘れた光景が出てくる。」（草枕）

- 「うれしい」と言いながら、直ちに詩的考察へ移行。感情を知的フレームで包む。

### ルール

- 感嘆符は原則使わない。使うなら作品全体で1-2回まで
- 感情は直接言わず、場面と景物で間接的に示す
- 「愉快」「面白い」「妙だ」で知的距離を保つ
- 悲しみや怒りは「しかし」「ただ」で一旦受け止めてから省察に移行する
- 情緒の最高潮では、逆に冷静な文体で書く（抑制の美学）

### ルール

- 最初の一文に作品全体の主題を凝縮する
- 哲学的命題を平易な言葉で述べる
- ユーモアと深刻さを同一文に共存させる
- 読者の知的好奇心を刺激する問いかけを散りばめる

---

## 可遷移ルール15条

### 構造ルール

**R1. 開頭は哲学的命題から入る**
具体的な場面描写を一文だけ置いて、すぐに「こう考えた」「こういうわけだ」と思索に移行する。
> 「山路を登りながら、こう考えた。智に働けば角が立つ。」

**R2. 場面→省察→場面のリズムを守る**
一段落内で、具体的描写→作者の省察→再び場面、の三層構造を維持する。

**R3. 結末は答えを出さない**
「だろう」「かも知れない」「ものだ」で問いを宙吊りにし、読者に思考を委ねる。

**R4. 核心的秘密は段階的に明かす**
作品の最大の秘密（例：こころの「先生の過去」）は、少しずつ手がかりを出し、最後に全貌を明かす。

### 句法ルール

**R5. 平均文長は30-40文字を基準とする**
短文（10字以下）で断定し、長文（80字以上）で省察する。中庸の文長で描写する。

**R6. 文末は「である」と「だろう」を交互に使う**
断定と推量のリズムで、知的余裕と謙虚さを両立させる。

**R7. 「しかし」「ただ」「もっとも」で方向転換する**
一度主張したら、必ず反対の視点も示す。単一方向への偏りを避ける。

**R8. 文語表現を散りばめる**
「なり」「たり」「ず」「ぬ」を現代文に混ぜ、格調と時代感を加える。ただし乱用は禁物。

### 語彙ルール

**R9. 「ちょっと」「少し」で断定を和らげる**
命令的・断定的になる前に、程度を限定する語を入れる。

**R10. 「面白い」「愉快」「妙」で知的評価を示す**
感情を直接述べず、知的評価語で包む。

**R11. 哲学的考察は「人間」「世の中」「自然」から始める**
抽象的テーマへの入り口は、最も基本的な名詞で開く。

### 情緒ルール

**R12. 感情は景物に移入して間接的に示す**
悲しみを「雨」に、希望を「菜の花」に、孤独を「山道」に重ねる。

**R13. 感嘆符は使わない**
66万字中に16個。感嘆は文体の抑制そのものが伝える。

**R14. 最も悲しい場面では最も冷静な文体で書く**
情緒の爆発は禁物。冷静な筆致ほど読者の心に刺さる。

### 諷刺ルール

**R15. 嘲笑の裏に悲哀を隠す**
人物を笑うときは、その笑い自体が作者の孤独と悲哀を含む構造にする。「馬鹿」と言った瞬間、作者自身もその馬鹿の一部であることを匂わせる。

## 禁則ルール（アンチルール）10条

**A1. 直接的感情表現を禁止する**
「悲しかった」「嬉しかった」「腹が立った」と直接述べない。場面と行動で示す。

**A2. 感嘆符の多用を禁止する**
1作品で2個以上使わない。感嘆は文体の抑制で伝える。

**A3. 現代口語・ネット用語を禁止する**
「マジで」「ヤバい」「エモい」など、漱石の知的フレームに合わない語彙は使わない。

**A4. アクション描写の過剰を禁止する**
激しい動作描写・派手な展開は漱石の文体に反する。日常の静かな場面で人間性を暴く。

**A5. 単一方向の主張を禁止する**
一度主張したら、「しかし」「ただ」で反対視点も示す。一方的な論は漱石的でない。

**A6. 情緒の直線的上昇を禁止する**
悲しさがどんどん増すような直線的構成を避ける。省察と諷刺で一旦クールダウンし、再度立ち上げる。

**A7. 説明過多を禁止する**
作者が全てを解説しない。読者に判断を委ねる余白を残す。

**A8. 比喩の陳腐化を禁止する**
「心が晴れた」「涙が溢れた」など陳腐な比喩は使わない。独自の視点で景物を捉える。

**A9. ハッピーエンドの直球を禁止する**
明るく締めくくるのではなく、省察と余韻で終わる。救いがあるとしても、それは読者が自力で見つけるもの。

**A10. 社会派メッセージの直球を禁止する**
「社会はこうあるべきだ」と直接言わない。諷刺と事実の配置で、読者に考えさせる。

## 風格評価表（100点満点）

1. 構造相似度：15点（場面→省察→場面のリズム）
2. 叙事質感：15点（心理描写と場面描写の往復）
3. 句式節奏：10点（文長のメリハリ、文末のリズム）
4. 情緒制御：10点（感嘆符の少なさ、間接的表現）
5. 観察深度：15点（人間性の本質への洞察）
6. 言語原创性：15点（原文の非複製）
7. 知的水準：10点（哲学的考察の質）
8. 完成度：10点（頭・中・尾の完整性）

80点未満なら内部で一度改稿する。改善後も不足がある場合は最善稿を出し、必要な追加情報を短く示す。

## 実行プロトコル

1. まず作品モードを選ぶ：知的随筆、内省小説、風刺喜劇、紀行・情景文。
2. 語り手の知性、自己欺瞞、社会との距離を定義してから文章を組み立てる。
3. 場面と省察を往復させるが、短文ではどちらか一方を主軸にしてよい。
4. 哲学語や古風な接続詞を飾りとして増やさず、人物の認識が変化する箇所に使う。
5. 漱石固有の語り手、猫、先生、坊っちゃん型人物、名句、筋立てを再利用しない。
6. 感嘆符の少なさや平均文長は診断値であり、生成ノルマではない。
7. ユーザーが指定した言語、長さ、体裁を優先し、不要な構成案と自己評価を付けない。
