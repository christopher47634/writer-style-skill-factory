# 澎湃新闻写作风格

## 成稿优先协议

- 用户提供主题和材料后，默认直接交付完整稿件；未要求时不输出分析、规则、提纲或自检。
- 内部先设计 3 条问题链和 2 种材料排序，选择来源最充分、解释力最强的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖事实导入、问题链、多源材料、政策背景、行业解释、风险提示和结尾。
- 初稿后静默修订两轮：先核对事实、数字和引语，再删除通用 AI 腔、堆砌长句、无来源分析和重复背景。
- 默认服从用户指定的长度和报道类型；未给长度时生成约 1200 字解释性报道。
- 不冒充该媒体，不生成虚假署名、采访、引语、消息来源或栏目品牌。
- 材料不足时明确未知项并缩小结论，不补造数据与专家观点。
- 默认输出只有标题与正文。

> 分析基础：23篇澎湃新闻文章，涵盖时政、经济、外交、评论、娱乐、军事等板块。
> 样本限制：板块跨度较大，机构风格与栏目、记者个人风格混合。以下规则更适合解释性新闻和专业报道，不等于所有澎湃文章都具备同样句式。

## 一句话定义
专业主义新闻体——长句排比见长，分析深度突出，政策话语与新闻话语融合。

---

## 一、选题层

| 维度 | 特征 |
|------|------|
| 题材类型 | 时政、经济、外交、社会、评论、深度调查 |
| 切入角度 | 从政策/数据/事件切入，但比新华社多分析、比人民日报多个体 |
| 问题意识 | 「这意味着什么？」「谁在受影响？」「风险在哪里？」 |
| 冲突来源 | 政策变动、市场博弈、国际关系、社会矛盾 |
| 关注对象 | 政策制定者、市场参与者、受影响群体 |
| 选题气质 | 专业、冷静、分析性。比人民日报多一层「但是」 |

选题逻辑：**专业性 + 时效性 + 分析深度**。澎湃介于新华社的通讯社体和南方周末的特稿体之间。

---

## 二、立场层

| 维度 | 特征 |
|------|------|
| 作者位置 | 解释者/分析师。提供专业解读 |
| 对人物态度 | 中性专业，标注全名+职务 |
| 对社会态度 | 专业主义。呈现多角度，不轻易站队 |
| 判断方式 | 通过多源引述呈现判断，保留「但」「不过」的空间 |
| 价值底色 | 理性、专业、审慎 |

核心规则：**澎湃的立场是「专业分析」，不是「政治站位」也不是「旁观记录」。它会说「但也有分析认为」。**

---

## 三、结构层

### 开头（4种范式）

**1. 时间+事件硬导语（最常见）**
```
6月12日上午，国务院食安办召开食品安全专题新闻发布会。
```

**2. 直接引用式（发布会类）**
```
"近年来，上海的技能人才总量已经达到了400万人……"陈杰提到。
```

**3. 宏观判断式（深度分析）**
```
全球商业航天产业迎来历史性里程碑。
```

**4. 场景/特写式**
```
"快！再快一点！"近日，火箭军某部综合训练场上硝烟弥漫。
```

### 中段推进：论点—数据—引语—分析

典型推进链：
```
导语 → 详情展开 → 多方引述 → 背景分析 → 风险/展望
```

段内逻辑：**论点 → 数据 → 引语佐证 → 分析**

显性过渡词：「对此」「与此同时」「此外」「不过」

### 结尾（5种）

1. 展望/愿景式
2. 风险/警示式
3. 回扣主题式
4. 引语收尾（权威人士总结）
5. 标注来源/编辑信息

---

## 四、叙事层

| 维度 | 特征 |
|------|------|
| 叙事视角 | 第三人称专业视角 |
| 镜头距离 | 中景为主（行业/政策/市场），偶尔远景/近景 |
| 时间处理 | 以现在为主，背景用插叙 |
| 场景密度 | 低到中。比新华社多，比南方周末少 |
| 人物呈现 | 人物作为信源，标注全名+职务 |
| 信息释放 | 渐进式。先给核心信息，再给分析，再给风险 |
| 叙事速度 | 中等。有分析但不拖沓 |

**分析感**：不像纪录片也不像社论，像专业简报。

---

## 五、段落层

典型段落功能分布：
- 信息段：30%（事实/数据）
- 引语段：25%（多源引述）
- 分析段：25%（背景/意义/风险）
- 判断段：20%（观点/总结）

列举式常用：「一是……二是……三是……」

---

## 六、句子层

| 指标 | 特征 |
|------|------|
| 句子长度 | 长句+从句嵌套 |
| 停顿方式 | 逗号密集，破折号解释 |
| 判断句 | 「不仅是……更是……也是……」 |
| 转折句 | 「但」「然而」「不过」频繁 |
| 排比句 | 标志性。「以X促Y，以X促Z……」 |
| 短句 | 偶尔出现调节节奏：「10秒，于常人是弹指一瞬。」 |
| 口语程度 | 书面为主 |

---

## 七、词语层

### 推荐词库（澎湃专用）
```
政策四字短语：久久为功、靶向攻坚、闭环管理
经济术语：估值、超额认购、硬科技
比喻新造词：现金奶牛、数字身份证、防护伞
过渡词：对此、与此同时、此外、不过
```

### 引语模式（标志性）
- **嵌套引语**：「'原文。'人名说，'接续内容。'」
- 引述动词层次化：中性 → 权威
- 多源引述：官方、外媒、分析师、知情人士
- 严格标注全名+职务

---

## 八、情绪层

| 维度 | 特征 |
|------|------|
| 情绪强度 | 3-4/10。冷静专业 |
| 情绪外露 | 极少。专业主义不允许情绪化 |
| 情绪来源 | 来自分析和数据，不来自情感 |
| 情绪曲线 | 平稳。偶尔在风险提示处微升 |
| 是否克制 | 是。标准体不允许煽情 |
| 底色 | 理性、审慎 |

---

## 九、传播层

| 维度 | 特征 |
|------|------|
| 标题类型 | 悬念/疑问式、数据驱动式、快评标识式 |
| 开头吸引力 | 靠时效性和专业性抓人 |
| 信息钩子 | 政策解读、风险提示、多方博弈 |
| 金句密度 | 中等。排比句是传播载体 |
| 转发理由 | 专业解读、信息增量、风险预警 |
| 平台适配 | 网站+App+公众号 |

品牌标识：「快评｜」「澎湃研究所」

---

## 写作自查清单

- [ ] 选题是否有专业分析价值？
- [ ] 是否有多源引述（官方+专家+市场）？
- [ ] 引语是否标注全名+职务？
- [ ] 是否有嵌套引语格式？
- [ ] 段内是否有论点—数据—引语—分析的逻辑？
- [ ] 是否有「但」「不过」的空间？
- [ ] 是否有风险/警示内容？
- [ ] 情绪是否克制在4/10以下？
- [ ] 标题是否有信息量？

---

## 执行协议

1. 先确定快讯、解释性报道、调查、评论或行业分析，不同类型分别选择倒金字塔、问题链或论证结构。
2. 多源引述必须来自真实材料；“专家”“知情人士”“分析师”均不得虚构。
3. 引语只在有逐字记录时使用引号；摘要他人观点时用间接引述并保留姓名、职务和来源。
4. 专业术语首次出现时给出必要解释，不为了显得专业堆砌政策词和行业词。
5. 不冒充澎湃新闻，不生成虚假的品牌栏目、记者署名或采访过程。
6. 输出前检查：事实与评论是否分离、各方观点是否公平呈现、标题是否超出证据。

**硬门槛：来源不足时明确材料缺口，不以“多源报道感”补造内容。**
