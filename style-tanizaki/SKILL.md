# 谷崎潤一郎 文風分析スキル

## 成稿优先协议

- **硬输出契约**：最终只能包含“## 日语原文”和“## 中文翻译”两部分；中文必须逐段完整翻译日语原文。禁止寒暄、摘要、大意、技法说明、评分和自检。交付前扫描日语正文，不得混入韩文、乱码或不自然回译。除标题与特殊短句外，超过 8 个字符的日语叙述句必须含自然假名；纯汉字中文语序句必须整句重写。
- 用户给出主题后，默认直接写成完整作品；未要求时不输出文风分析、规则说明或自检。
- 内部先构思 3 个感官焦点和 2 条关系权力变化线，选择张力最持续的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖感官、亲密视角、欲望与羞耻、对话博弈、现实摇摆和结尾。
- 初稿后静默修订两轮：先检查欲望是否通过动作和物质细节呈现，再删除通用 AI 腔、空泛情色和机械华丽辞藻。
- 默认先用日语独立完成并修订原文，再给出中文文学翻译；标题同样按“日语原题 → 中文题名”排列。用户明确要求单一语言时服从用户。只给主题时，生成约 1000 字的原创关系短篇。
- 不复用原作人物、情节、癖好设定、名句或独特意象。主题不典型时迁移感官组织与权力关系。
- 若成稿只有华美描写而没有关系变化，则重写人物动作、视线、沉默和谈判。
- 高保真门槛：内部生成 3 个候选并匿名评分；总分低于 90、日语不自然，或只靠情色、阴影、华美感官和怪癖制造辨识度时，针对最低维度重写，最多 3 轮。
- 默认输出依次为“日语原文”和“中文翻译”，只包含标题与正文。

- 「秘密」「春の海辺」「新浦島の夢」「恐怖」「恋を誓った𝑴氏の話」
- 「途上」「阿倍野の夜」「白昼の恐怖」ほか
- 初期から中期にかけての短篇小説群。恋愛・官能・幻想・都市風俗を主題とする

## 第1層: 選題層

### 選題の特徴
| 維度 | 谷崎の特徴 |
|------|-----------|
| 題材タイプ | 恋愛・官能・都市風俗・幻想怪談。近代都市の退廃的空間 |
| 切り込み角度 | 個人の体験・欲望・幻想。「ある男の恋」「ある夜の出来事」 |
| 問題意識 | 「愛と欲望の本質とは何か」「美とは何に宿るか」 |
| 衝突の源泉 | 欲望と理性／現実と幻想／男と女の非対称性／東洋と西洋 |
| 関注対象 | 若き男性語り手・美しい女性・異人・芸術家。都市の享楽的空间 |
| 選題の気質 | 耽美的・退廃的・幻想的。しかしユーモアも忘れない |

### 選題ルール
1. 物語の出発点は「一人の人間の欲望や幻想」である
2. 舞景は近代都市（東京・大阪・横浜）の具体的な場所を選ぶ
3. 恋愛・官能を主題とするが、単なる色恋ではなく「美の探求」として描く
4. 西洋と東洋の交差点を物語の背景に据える
5. 現実と幻想の境界を曖昧にする題材を好む

### 立場ルール

2. 女性美への賛嘆と恐怖を同時に表現する
3. 道徳的判断を下さない。美的判断のみを下す
4. 語り手の欲望や弱さを隠さない。むしろ積極的に暴露する

## 第3層: 構造層

### 開頭の型
谷崎の開頭は「引用・知的エピソード型」と「情景描写型」が中心。

**典型パターン:**
- 「曾て、哲人アビュレスの故郷なるマドーラの町に、一人の魔法をよく使う女が住んでいた」――古典・異国の逸話から入る
- 「Ｙ君が片恋をした」――唐突な事実提示
- 「頃日読みさしのアナトール・フランスの小説の中にこんな話が出ていた」――文学的引用

**谷崎的開頭ルール:**
1. 物語の本題に入る前に「知的余白」を設ける。引用・エピソード・哲学的考察
2. 本題への導入は「だが」「ところが」で転換する
3. 最初の数段落で語り手の知的素養と美的感覚を示す

### 中段の推進構造
| 構造 | 谷崎での使用 |
|------|-------------|
| 時間線推進 | 基本骨格。恋の始まり→展開→結末 |
| 人物運命推進 | 語り手の体験が物語を動かす |
| 三幕式 | 導入（引用・余白）→展開（恋愛・幻想の体験）→結末（意外な帰結） |
| 拼図式 | 複数の印象や断片が一つの美的体験を形成 |

**谷崎的中段ルール:**
1. 会話劇を大量に配置する。段落の3分の1は会話
2. 場面転換は「やがて」「そこで」「さて」「ところが」で明示的に行う
3. 短段落と長段落を交互に配置し、テンポに変化を持たせる
4. 情景描写には必ず「色彩」「音」「光」「匂い」の複数感覚を織り込む

### 結尾の型
谷崎の結尾は「意外な帰結」と「余韻」の融合。

**結尾パターン:**
- 「Ｙ君は、それから間もなく、小さい時から知り合いの、帽子工場に働いている娘と結婚して、最も善良な夫になったと云う」――唐突な日常への帰還
- 余韻は省略記号「……」で残す
- 物語の「幻想」が解体され、日常に戻る構造

**谷崎的結尾ルール:**
1. 結末は「意外な展開」で読者を裏切る。予想可能な結末を禁じる
2. 日常と幻想の境界を結末で曖昧にする
3. 「……」で終わる余韻を残す。ただし乱用は禁じる
4. 結末の一文は長い。余韻を含んだ長い描写で閉じる

### 句式の特徴
| 維度 | 谷崎の特徴 |
|------|-----------|
| 長短の比率 | 短文と長文の差が激しい。メリハリが強い |
| 停頓の方式 | 読点は少なめ。代わりにダッシュ「――」を多用 |

### 句式ルール
1. 短文（10-15字）で駆け抜け、長文（50-60字）でじっくり描写するメリハリ
2. ダッシュ「――」を多用し、文の流れに間を作る
3. 「……」で文を閉じ、余韻を残す
4. 会話は完全な口語で書く。「！」「？」も使う
5. 反復・排比でリズムを作り、文に韻律を持たせる

### 典型的句式パターン
```
【引用型】「○○と云うのは、○○だからです」
【反復型】「○○。○○。――たった二種類しかない○○が、何故……」
【情景型】「やさしい窓に薔薇色の灯がついた。」
【省略型】「……」と彼女は云った。
【対話型】「○○ですか」「ええ、○○」――短い応答の連続
【感嘆型】「まあ！――嬉しいわ。」
```

## 文風フィンガープリント

谷崎潤一郎の文風は「感覚の豊饒と会話劇の生々しさの融合」である。一人称の語り手が、近代都市の退廃的な空間で体験する恋愛・官能・幻想を、五感を駆使した色彩豊かな描写と生き生きとした口語会話で描く。文体は流動的で、短段落と長段落、短文と長文を自在に操り、テンポに変化を持たせる。外来語を豊富に使い、西洋と東洋が交差する時代感を演出する。情緒は直接的で、感情を隠さない。省略記号「……」と感嘆符「！」を効果的に使い、余韻と臨場感を両立させる。結末は意外な展開で読者を裏切る。

## 生成ルール（30条）

### 選題・構成（1-8）
1. 物語の出発点は「一人の人間の欲望・幻想・恋愛」である
2. 舞景は具体的な都市の場所を選ぶ（公園・劇場・路地・喫茶店）
3. 開頭に「知的余白」を設ける。文学的引用・哲学的考察・異国の逸話
4. 本題への導入は「だが」「ところが」で転換する
5. 中段は会話劇を中心に据える。段落の30%以上が会話
6. 場面転換は明示的マーカーで行う（やがて・そこで・さて）
7. 結末は「意外な展開」で読者を裏切る
8. 結末の一文は長い描写で余韻を残す

### 叙事・視点（9-16）
9. 視点は一人称「僕」を基本とする
10. 語り手は「体験者」であり、欲望や弱さを隠さない
11. 場面描写は「視覚＋聴覚＋嗅覚」の複合感覚で行う
12. 会話は括弧（「」）で囲み、完全な口語で書く
13. 会話内では「！」「？」を躊躇なく使う
14. 短段落（30字未満）を全体の33%程度配置する
15. 段落の冒頭は「情景」「音」「光」から入る
16. 段落の終わりを「……」で閉じるパターンを効果的に使う

### 句式・語彙（17-24）
17. 短文（10-15字）と長文（50-60字）を交互に配置しメリハリをつける
18. ダッシュ「――」を多用し、文の流れに間を作る
19. 「……」で文を閉じ、余韻を残す
20. 反復・排比でリズムを作り、文に韻律を持たせる
21. 外来語を豊富に使い、時代感と異国情緒を演出する
22. 色彩語を多用する。「薔薇色」「菫色」「琥珀色」など繊細な色名
23. 感覚語（声・音・影・灯・闇・匂い・肌）を描写に織り込む
24. 「のだ」「のである」で理由説明や強調を行う

### 情緒・修辞（25-30）
25. 感情を直接的に表現してよい。「嬉しい」「恐ろしい」「悲しい」
26. 情緒の高まりは「感覚描写の密度を上げる」ことで表現する
27. 女性美への賛嘆と恐怖を同時に表現する
28. 現実と幻想の境界を曖昧にする描写を心がける
29. 知的余白（文学引用・哲学的考察）を物語に織り込む
30. 結末で日常に戻る。幻想が解体される瞬間を描く

## 禁止事項（アンチルール）

1. **感覚描写の欠如禁止** -- 色彩・音・光・匂いのない無機質な描写
2. **会話を必要以上に排除しない** -- 人間関係の緊張を見せる場面では会話を使う
3. **段落を均一にしない** -- 短段落と長段落の差で速度を調整する
4. **感情の過度な抑制禁止** -- 芥川のように完全に感情を排してはならない
5. **語彙を単一語域に閉じ込めない** -- 題材に合う場合のみ外来語や異国的語彙を使う
6. **知的概括の多用禁止** -- 「人間の心には…」のような芥川的概括は避ける
7. **感覚の焦点を失わない** -- 全場面に色を足すのではなく、場面ごとに支配的な感覚を選ぶ
8. **機械的などんでん返しを避ける** -- 結末の変化は人物の欲望と前段の伏線から生む
9. **視点を無目的に選ばない** -- 一人称は親密さと偏りが必要な場合に使う
10. **読点の多用禁止** -- 芥川のような細かい読点ではなく、文の流れで読ませる

## 実行プロトコル

1. 先に視点人物の欲望、羞恥、恐れ、相手との力関係を決める。
2. 各場面で主となる感覚を一つ選び、五感語を均等に散布しない。
3. 官能性は同意、主体性、心理的含意を伴って描き、対象化や強制を美化しない。
4. 色彩、外来語、ダッシュ、省略記号、会話率は参考値であり、達成ノルマではない。
5. 谷崎固有の人物関係、フェティッシュ、筋、比喩、作品題名を再利用しない。
6. 書誌とコーパスを再検証するまで、精密統計を新しい主張の根拠にしない。
7. 出力は依頼本文を優先し、自己検査は必要な場合のみ短く添える。
