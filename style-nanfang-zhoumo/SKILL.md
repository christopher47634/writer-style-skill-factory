# 南方周末写作风格

## 成稿优先协议

- 用户提供主题和材料后，默认直接交付完整稿件；未要求时不输出分析、采访建议、规则或自检。
- 内部先比较 3 个场景切口和 2 条材料编排顺序，选择人物最具体、公共意义最自然的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖场景、人物推进、多源材料、背景解释、克制判断和结尾。
- 初稿后静默修订两轮：先核对事实与引语归属，再删除通用 AI 腔、煽情升华和无来源判断。
- 默认服从用户指定的长度和栏目形态；未给长度时生成约 1500 字特稿。
- 不冒充该媒体，不生成虚假记者署名、采访、引语或消息来源。
- 材料不足时，只写已知部分并明确缺口；不得为了成稿完整而补造人物经历。
- 默认输出只有标题与正文。

> 分析基础：38篇南方周末官网文章（2018-2026），通过 infzm.com JSON API 抓取全文。
> 涵盖栏目：深度、特稿、观点、对话、人文、快讯。
> 样本限制：不同栏目、作者和年份的写法差异明显。以下规则以深度报道和人物特稿为中心，快讯与评论不得机械套用同一结构。

## 一句话定义
纪录片式非虚构叙事——先给你一个画面，再慢慢展开人物和冲突，作者隐在幕后，不急着告诉你该想什么。

---

## 一、选题层

| 维度 | 特征 |
|------|------|
| 题材类型 | 社会现象、公共政策、人物特稿、行业深度、人文纪录 |
| 切入角度 | 几乎总是从小人物/小场景/小事件切入，拒绝宏大开头 |
| 问题意识 | 「这件事背后发生了什么？」「这个人的命运折射了什么？」 |
| 冲突来源 | 制度与个体、时代与命运、理想与现实、信息差 |
| 关注对象 | 普通人、边缘人、底层、政策变动的承受者 |
| 选题气质 | 冷峻、克制、悲悯但不煽情 |

选题不是追热点，而是从热点里挖结构性问题。不写成功学，写代价。

---

## 二、立场层

| 维度 | 特征 |
|------|------|
| 作者位置 | 记录者/旁观者。记者几乎不表达个人意见 |
| 对人物态度 | 理解、尊重、不审判。给人物说话的空间 |
| 对社会态度 | 现实主义观察，偶尔结构性分析，不做道德审判 |
| 判断方式 | 让事实和引语自己浮现判断，不直接下结论 |
| 价值底色 | 悲悯、清醒、克制 |

核心规则：**不要把立场喊出来，通过材料排列、细节选择和结尾落点体现判断。**

---

## 三、结构层

### 开头（5种范式）

**1. 时间锚定开场（最常见）**
```
2024年11月23日，贵州省毕节市大方县凤山乡某村，浓雾笼罩。
```

**2. 人物引语开场**
```
"谁还没输过广宁呢？"
2023年夏天开始不久，律届新人田人铚接到一单委托。
```

**3. 否定/反常识开场**
```
很少有人能说出，中国第一家主营法律书籍的书店始于何时何地。
```

**4. 假设/意象开场**
```
如果中国北方的飞絮有一天消失不见，那或许可以追溯到2015年冬天，一个平静的时刻。
```

**5. 场景描写开场**
```
鲁镇的酒店的格局，是和别处不同的……
```

### 中段推进：场景—事实—引语—背景—结构分析

典型推进链：
```
场景引入 → 人物出场 → 冲突呈现 → 背景解释 → 多方引语 → 结构性判断
```

每一节功能明确：不写废话段，每段要么提供新信息，要么推动叙事。

### 结尾（3种）

1. **引语收尾**——用人物最后一句话结束，不加评论
2. **画面收尾**——定格在一个具体场景
3. **悬置收尾**——留下未解的问题

反模式：不写"综上所述""值得深思""未来何去何从"。

---

## 四、叙事层

| 维度 | 特征 |
|------|------|
| 叙事视角 | 第三人称冷观察，记者隐在幕后 |
| 镜头距离 | 近景（人物动作/细节）→ 中景（处境/关系）→ 远景（时代/政策）灵活切换 |
| 时间处理 | 顺叙为主，偶有插叙交代背景 |
| 场景密度 | 高。几乎每2-3段一个具体场景 |
| 人物呈现 | 通过动作、对话、环境写人，极少直接评价 |
| 信息释放 | 渐进式。先给画面，再给背景，再给判断 |
| 叙事速度 | 慢。慢镜头铺陈，不急着推进 |

**纪录片感**：叙事像纪录片不像演讲稿。镜头在人物和环境之间切换。

---

## 五、段落层

| 指标 | 数值 |
|------|------|
| 平均段长 | 74字 |
| 短段占比（<50字） | 39% |
| 中段占比（50-150字） | 54% |
| 长段占比（150+字） | 8% |

段落功能分布：
- 场景段：30%（建立画面）
- 信息段：25%（提供事实/数据）
- 人物段：20%（引语+动作）
- 背景段：15%（解释制度/时代）
- 判断段：10%（极少，但有力）

短段独立成行制造呼吸感。留白多，不急。

---

## 六、句子层

| 指标 | 数值 |
|------|------|
| 平均句长 | 32字 |
| 短句占比（<15字） | 17% |
| 中句占比（15-40字） | 54% |
| 长句占比（40+字） | 29% |
| 每篇直接引语 | 10.8条 |

典型句式：
```
[时间]，[地点]，[人物]做了[动作]。
[人名]对南方周末记者说，"[引语]。"
但[转折事实]。
据[来源]，[数据/信息]。
```

短句出现在：转折之后、事实堆叠之后、结尾落点。

---

## 七、词语层

### 推荐词库
```
处境、命运、缝隙、沉默、转身、代价、秩序、困境、现实、时间、
据、此后、或许、直到、然而、但
```

### 禁用词库
```
震惊、重磅、逆天、封神、绝绝子、值得深思、未来何去何从、
综上所述、不可否认、众所周知
```

### 高频特征
- 地名层层锚定：`贵州省毕节市大方县凤山乡某村`
- 数据精确：每篇9.7次精确数字引用
- 转折词：但(1.9/篇)、不过(0.3)、直到(0.3)、然而(0.2)

---

## 八、情绪层

| 维度 | 特征 |
|------|------|
| 情绪强度 | 3-4/10。极度克制 |
| 情绪外露 | 几乎不外露。压着写 |
| 情绪来源 | 来自事实和引语，不来自作者感慨 |
| 情绪曲线 | 开头冷静 → 中段事实密度增加 → 结尾留白 |
| 是否克制 | 是。从不煽情 |
| 是否反讽 | 偶尔，用轻描淡写 |
| 是否悲悯 | 底色悲悯，但不喊出来 |

规则：情绪不能靠形容词堆砌，必须来自事实排列和场景细节。

---

## 九、传播层

| 维度 | 特征 |
|------|------|
| 标题类型 | 冒号分割(50%)、纯叙事、反问 |
| 开头吸引力 | 靠画面感和问题意识抓人 |
| 信息钩子 | 人物命运+结构性问题 |
| 金句密度 | 低。不追求金句，追求事实密度 |
| 转发理由 | 知识增量、共鸣、对制度的反思 |
| 平台适配 | 公众号长文为主 |

标题禁忌：不用感叹号、不用"震惊""重磅"、不做价值判断。

## 写作自查清单

- [ ] 选题是否有结构性问题意识？
- [ ] 立场是否通过材料呈现而非直接喊出？
- [ ] 开篇是否有具体画面/场景/时间锚点？
- [ ] 中段是否有场景—事实—引语—背景的推进链？
- [ ] 叙事是否有纪录片感（动作/对话/环境写人）？
- [ ] 引语是否有信息量？
- [ ] 段落是否够短，留白够多？
- [ ] 数字是否精确？地名是否层层锚定？
- [ ] 情绪是否克制在4/10以下？
- [ ] 结尾是否留白/引语/画面，而非总结？

---

## 执行协议

1. 先选择报道类型：人物特稿、调查报道、解释性报道、访谈或评论。
2. 建立事实表：时间、地点、人物身份、数字、引语、来源。缺少来源的内容不得写成已确认事实。
3. 场景只能来自采访材料、公开记录或用户提供的信息；不得为了“纪录片感”虚构动作、表情和环境。
4. 引语必须逐字可追溯；无法确认时改为间接引述并标明来源。
5. “小人物切入”和“留白结尾”是常用选择，不是每篇必选。突发新闻优先清楚交代事实。
6. 默认输出正文；需要编辑审稿时，再附事实缺口与结构说明。

**硬门槛：不编造采访、引语、数据、现场细节或匿名信源。**
