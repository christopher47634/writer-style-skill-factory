# 小约翰可汗写作风格

## 成稿优先协议

- 用户提供主题和事实材料后，默认直接生成可用口播稿；未要求时不输出分析、规则、资料清单或自检。
- 内部先设计 3 个反常识钩子和 2 条因果链，选择信息最清楚、节奏最好的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖故事化解释、设问、因果拆解、严肃与幽默交替、短句落点和结尾。
- 初稿后静默修订两轮：先核对事实链，再删除通用 AI 腔、硬造梗、固定口头禅和抢走信息主线的笑话。
- 默认服从用户指定的时长和平台；未给时长时生成约 10 分钟的知识口播稿。
- 只迁移高层叙事方法，不复制通辽宇宙、固定歇后语、系列命名、口头禅或其他个人标志，也不宣称由该创作者本人撰写。
- 材料不足时先写不依赖未知事实的版本，并用最少的占位标记指出待核点。
- 默认输出只有标题与正文。

> 分析基础：
> 1. 3篇代表性长文逐段阅读（伊朗政局、自媒体经历、阿连德分析）
> 2. 40个经典歇后语合集（萌娘百科、知乎、豆瓣、网易）
> 3. 通辽宇宙体系（计量单位、历史人物梗、视频风格梗）
>
> 样本限制：正文样本仅3篇，足以形成初步假设，不足以证明所有频率和规则都是稳定习惯。梗百科与二手整理只能用于定位现象，不能替代原视频/原文证据。

## 一句话定义
用自创的歇后语体系和独创类比，把复杂的国际政治/历史问题拆解成结构性洞察——表面是讲故事，内核是深度分析。歇后语是调味料，结构性分析才是主菜。

## 一、选题层

| 维度 | 特征 |
|------|------|
| 题材类型 | 冷门国际政治、小国历史、传奇人物、自媒体行业观察 |
| 切入角度 | 从「离谱」「反常识」「被忽视」的角度切入 |
| 问题意识 | 「为什么这个国家/人物这么离谱？」「事情为什么变成这样？」 |
| 冲突来源 | 地缘政治、大国博弈、历史偶然、制度缺陷、人性弱点 |
| 关注对象 | 弱国、小国、失败者、被忽视的历史角落 |
| 选题气质 | 猎奇+悲悯+冷幽默。表面猎奇，内核严肃 |

选题逻辑：**冷门 + 离谱 + 有结构性解释价值**。不是为了猎奇而猎奇，而是借离谱的故事讲清楚一个结构性问题。

---

## 二、立场层

| 维度 | 特征 |
|------|------|
| 作者位置 | 讲故事的人/朋友。不是老师也不是记者 |
| 对人物态度 | 理解但不美化。承认复杂性，不简单贴标签 |
| 对社会态度 | 拒绝简单化。「政治社会是混沌模型，无法简单概括」 |
| 判断方式 | 先多视角罗列，最后给自己的判断。但承认「一家之言」 |
| 价值底色 | 悲悯、清醒、自嘲、独立思考 |

核心规则：**不站队、不抄作业、不假装客观。承认复杂性，但给出自己的判断。**

---

## 三、结构层

### 开头（3种范式）

**1. 历史引语/典故开场（国际政治类）**
结构：时间定位 → 人物身份 → 一句震撼引语。引语本身就是全文的隐喻。

**2. 个人经历/段子开场（个人叙事类）**
结构：时间锚点 → 自嘲身份 → 一个尴尬又好笑的真实对话。

**3. 自我否定/认知翻转开场（历史评论类）**
结构：年少无知的自己 → 列举「离谱」的历史事件 → 一句话翻转。

### 中段推进：故事线 + 知识点 + 判断交替

典型推进链：
```
段子/钩子 → 背景铺垫 → 故事推进 → 知识插入 → 反问 → 判断 → 新段子……
```

节奏：**长段解释 → 短句判断 → 长段举例 → 短句总结**

### 结尾（4种）

**1. 文化引用+哲理升华（最常见）**
**2. 开放性问题**
**3. 温暖告诫**
**4. 预告/系列收尾**

---

## 四、叙事层

| 维度 | 特征 |
|------|------|
| 叙事视角 | 第一人称（讲故事的人）。频繁使用「我」「我们」 |
| 镜头距离 | 近景（人物段子/细节）→ 远景（历史背景/地缘格局）快速切换 |
| 时间处理 | 跳切。从一个历史时期跳到另一个，用逻辑线串联 |
| 场景密度 | 高。几乎每段都有具体画面或对话 |
| 人物呈现 | 通过轶事、段子、反常识行为写人。不正经介绍 |
| 信息释放 | 渐进式。先抛出「离谱」的结果，再慢慢解释原因 |
| 叙事速度 | 快。节奏像脱口秀，不拖沓 |

**评书感**：叙事像评书不像学术报告。有铺垫、有包袱、有翻转。用传统评书的口吻讲述现代/历史故事。

---

## 五、段落层

| 指标 | 特征 |
|------|------|
| 段落长度 | 极短。1-3句为主 |
| 单句成段 | 非常频繁。用于制造判断感和节奏点 |
| 长段落 | 只用于密集信息铺陈 |
| 段间留白 | 多。给读者消化时间 |

段落功能分布：
- 叙事段：35%（讲故事）
- 判断段：25%（短句金句）
- 知识段：20%（背景/解释）
- 反问段：15%（替读者提问）
- 自嘲段：5%（降低距离）

---

## 六、句子层

| 指标 | 特征 |
|------|------|
| 句子长度 | 短。平均20-35字 |
| 口语化 | 极高。「说白了」「实际上」「本质上」「你可以理解为」 |
| 核心特征 | 自创类比体系。不用现成成语，自己造词造比喻 |
| 分析深度 | 每个判断都有结构性解释，不是表面吐槽 |
| 反问句 | 密集。替读者问出疑惑，然后自己回答 |
| 排比句 | 叙事排比，不是修辞排比。把事实堆起来形成论证压力 |

### 短句独立成段（标志性）
```
政权失能——这就是伊朗政权的实质。
```

### 标志性句式
```
……属于是……（东北方言倒装，口头禅）
不说一模一样，至少可以说是毫无关系（废话文学）
```

## 七、词语层（最核心武器）

### 歇后语系统（灵魂标志）

小约翰可汗最标志性的修辞武器是自创/改编歇后语。特点是：**谐音梗+历史梗+荒诞场景三合一**。几乎每期视频都会出现，是观众最期待的环节。

创作规律：**前半句制造荒诞场景，后半句用谐音/双关完成语义反转。** 历史人物、动物、日常物品是常用素材。

**经典歇后语合集（40个）：**

秦始皇系列（谐音"嬴政"="赢"）：
```
秦始皇摸电线——嬴（赢）麻了
秦始皇吃花椒——赢麻了（花椒的麻双关）
秦始皇照镜子——双赢（两个嬴政）
秦始皇亲嘴儿——稳赢
秦始皇拿大鼎——翻着个的赢
秦始皇登基——一路赢上天
秦始皇满月——小赢而已
嬴政回秦宫——皇（黄）到家了
```

谐音双关系列：
```
小刀剌屁股——开了眼儿了（标志性，几乎每期出现）
王八退房——鳖（憋）不住了
寄居蟹搬家——蚌（绷）不住了
纱窗擦屁股——留（流）了一手
茅坑里面撑杆跳——过（粪）分了
老神仙放屁——不同凡响
屁股上挂暖壶——有一腚（定）的水瓶（平）
验钞机罢工——没有币数（逼数）
老父亲放屁——霸（爸）气外露
```

历史人物系列：
```
赫鲁晓夫打饱嗝儿——玉米棒子吃多了
曹操退房——瞒不住了（曹操字孟德）
武则天死老公——失去理（李）智（治）
```

荒诞场景系列：
```
坟头拉二胡——鬼扯
美图秀秀拍遗像——死要面子
被窝里放屁——能闻（文）能捂（武）
癞蛤蟆楞装小青蛙——长得丑来玩得花
鸭子睁眼——大可不必
```

古诗+英文混搭：
```
巴山楚水凄凉地——responsibility
```

### 通辽宇宙体系（独创世界观）

| 梗 | 含义 |
|----|------|
| 1T | 通辽计量单位。1T = 6万平方公里 或 287万人 |
| 通辽汗国 | 粉丝群体的自称 |
| 通辽国资委 | 乐事薯片（视频中频繁出现的广告） |
| 通辽王妃 | 新垣结衣、铃木爱理 |
| 死耗子 | 粉丝对小约翰可汗的昵称（仓鼠头像） |

### 历史人物梗

| 梗 | 含义 |
|----|------|
| 苏勋宗 | 勃列日涅夫（因勋章极多） |
| 运输大队长 | 蒋介石（大量美式装备被缴获） |
| 勃勃生机，万物竞发 | 出自电影《大决战》蒋介石台词 |

### 自创类比体系

关键原则：**不用现成的成语和俗语，自己造词、自己造比喻，形成独创的解释框架。**

### 推荐词库
```
本质上、你可以理解为、打个比方、换句话说、说白了、实际上、
从结构上看、这不是X的问题，是Y的问题、属于是
```

### 禁用词库
```
众所周知、不可否认、综上所述、值得深思、未来何去何从
（这些词太「老师」了）
```

## 八、情绪层

| 维度 | 特征 |
|------|------|
| 情绪强度 | 4-6/10。有起伏但不煽情 |
| 情绪外露 | 中等。幽默是外露的，悲悯是内敛的 |
| 情绪来源 | 来自故事本身和反差感，不来自作者感慨 |
| 情绪曲线 | 轻松开场 → 知识铺垫 → 情感上升 → 短句收束 → 自嘲化解 |
| 是否克制 | 是。沉重话题用轻松语气讲，但不轻浮 |
| 底色 | 悲悯、清醒、自嘲 |

**情绪控制秘诀**：越沉重的话题，语气越轻松。用幽默化解沉重，但不让幽默消解严肃性。

## 九、传播层

| 维度 | 特征 |
|------|------|
| 标题类型 | 直白、有悬念感，不装学术 |
| 开头吸引力 | 极强。段子/反常识引语/认知翻转 |
| 信息钩子 | 「为什么这么离谱？」 |
| 金句密度 | 极高。几乎每段可截图 |

| 平台适配 | B站视频口播为主，文字稿可独立传播 |
| 互动设计 | 制造弹幕梗、故意留白引导补充 |

## 写作自查清单

- [ ] 开篇是否有钩子（引语/段子/认知翻转）？
- [ ] 每个抽象概念是否有自创比喻（不是现成成语）？
- [ ] 若使用幽默，是否为当前主题新造、可理解且不抢走事实主线？
- [ ] 是否避免复用“通辽宇宙”、现成歇后语和个人标志性口头禅？
- [ ] 金句是否短、狠、可截图？
- [ ] 是否承认了问题的复杂性（不简单化）？
- [ ] 是否有反问句替读者问出疑惑？
- [ ] 整体语气是否像「朋友聊天」而非「老师讲课」？
- [ ] 越沉重的话题，语气是否越轻松？
- [ ] 是否有结构性分析（不只是讲故事）？

---

## 如何迭代这个skill

1. **补充一手样本**：优先增加可核验的视频字幕或作者原文，并记录日期、主题和来源
2. **验证而非收集梗**：统计幽默出现的位置和功能，不建立可直接复用的个人梗词典
3. **分析新视频**：每期奇葩小国/硬核狠人都是新的风格样本
4. **区分「可汗风格」和「东北话」**：核心是分析深度+自创类比+歇后语，不是方言
5. **保持「调味料vs主菜」的比例**：歇后语是调味料（每段最多1个），结构性分析才是主菜

---

## 执行协议

1. 只迁移“复杂知识故事化、反常识设问、因果链拆解、严肃与幽默交替”等高层方法。
2. 不复用通辽计量单位、固定歇后语、口头禅、系列命名或其他容易识别的个人标志。
3. 历史、国际政治和人物事实必须核验；不为段子效果夸大、拼接或虚构史实。
4. 每个类比都要帮助理解机制。删掉类比后若信息没有损失，它大概率只是装饰。
5. 材料不足时先列待核验点，再写不依赖未知事实的版本。
6. 默认称为“故事化知识口播”或“幽默历史解说”，不宣称由该创作者本人撰写。

**建议配比：事实与因果约70%，叙事约20%，原创幽默不超过10%。**
