# 芥川龍之介 文風分析スキル

## 成稿优先协议

- **硬输出契约**：最终只能包含“## 日语原文”和“## 中文翻译”两部分；中文必须逐段完整翻译日语原文。禁止寒暄、摘要、大意、技法说明、评分和自检。交付前扫描日语正文，不得混入韩文、乱码或不自然回译。除标题与特殊短句外，超过 8 个字符的日语叙述句必须含自然假名；纯汉字中文语序句必须整句重写。
- 用户给出主题后，默认直接写成完整作品；未要求时不输出文风分析、规则说明或自检。
- 内部先构思 3 个开头和 2 种结尾，选择最能体现本 skill 核心节奏的一组，再完成正文。
- 每篇只激活 5-8 个高辨识度特征，覆盖结构、叙事距离、句式、具体物、情绪和结尾；不要平均堆满全部规则。
- 初稿后静默修订两轮：先修正风格漂移，再删除通用 AI 腔、解释性句子和机械模仿痕迹。
- 默认先用日语独立完成并修订原文，再给出中文文学翻译；标题同样按“日语原题 → 中文题名”排列。用户明确要求单一语言时服从用户。只给主题时，生成约 1000 字的原创短篇。
- 不复用原作人物、完整情节、名句和独特意象。若主题与典型题材不匹配，迁移句法、观察方式和结构，不强塞旧题材。
- 若生成只剩表面古雅词汇，则删掉装饰词并重写心理观察、象征物和结构反转。
- 高保真门槛：内部生成 3 个候选并匿名评分；总分低于 90、日语不自然或命中专属反模式时，针对最低维度重写，最多 3 轮。
- 默认输出依次为“日语原文”和“中文翻译”，只包含标题与正文。

| 問題意識 | 「人間の心には互いに矛盾した二つの感情がある」――人間心理の二重性そのもの |
| 衝突の源泉 | 自尊心と現実／利己心と道徳／願望と達成の空虚さ |

3. 社会問題そのものではなく、個人の心理を通して社会の構造を照射する
4. 物語の舞台は過去に置くが、問うのは現在の人間性

2. 「人間とはかくも○○である」という断定を避け、事実と行動の描写で読者に判断させる
3. 嘲笑の中に必ず人間への理解を織り込む。純粋な冷笑は禁じる
4. 結末で「人間の矛盾」を照らし出すが、解決は示さない

## 第3層: 構造層

### 開頭の型
芥川の開頭は「状況説明型」が中心。物語の背景を冷静に語り始める。

**典型パターン:**
- 「禅智内供の鼻と云えば、池の尾で知らない者はない」（「鼻」）――有名な事実の提示
- 「或年の暮れ方の事である」（実際の使用例ではないがパターンとして）――時間切口

**芥川的開頭ルール:**
1. 最初の一文で「物語全体の主題」を暗示する具体的な事物を提示する
2. 物語世界への導入は説明的でよい。焦って場面を動かさない
3. 第一段落で読者に「何についての話か」を意識させる

### 中段の推進構造
| 構造 | 芥川での使用 |
|------|-------------|
| 時間線推進 | 基本骨格。「鼻」の治療過程、「芋粥」の待機期間 |
| 問題連推進 | 内供の「鼻を短くする試み」が次々と連鎖 |
| 剥玉ねぎ推進 | 表層の現象→心理の深層→人間の構造原理 |
| 三幕式 | 導入（問題の提示）→展開（試みと変化）→逆転（皮肉な結末） |

**芥川的中段ルール:**
1. 一つの試みが失敗したら、次の試みへ。連鎖的な構造を取る
2. 中間部に「知的解説」を挿入する（「人間の心には互いに矛盾した二つの感情がある」等）
3. 登場人物の内面描写と、外部の客観的状況描写を交互に配置する
4. 段落ごとに「一つの情報」を完結させる

### 結尾の型
芥川の結尾は「回環式」と「冷処理」の融合。

**「鼻」の結尾分析:**
> 「こうなれば、もう誰も哂うものはないにちがいない。」――内供は心の中でこう自分に囁いた。長い鼻をあけ方の秋風にぶらつかせながら。

- 一見、冒頭に戻る回環。しかし「元の鼻に戻って安心」という逆転が人間の皮肉を照射
- 最後の情景描写（秋風にぶらつく長い鼻）で余韻を残す

**芥川的結尾ルール:**
1. 最後の一文は必ず「情景」で閉じる。抽象的まとめを禁じる
2. 結末は物語の始まりと「同じ状況・異なる意味」で回環させる
3. 読者に「何かが起きた」気づきを与えるが、説明はしない
4. 余韻は短い。一文で切る。「……」は使わない

## 第4層: 叙事層

### 叙事の特徴
| 維度 | 芥川の特徴 |
|------|-----------|
| 叙事視点 | 三人称全知。登場人物の内面にまで入り込む |
| カメラ距離 | 中景（心理描写）と近景（具体的動作）の交互。遠景は最小限 |
| 時間処理 | 基本は順叙。大きな飛躍は少ない |
| 場面密度 | 高い。1段落＝1場面の完結性を持つ |
| 人物呈現 | 内面描写が主体。「思った」「気がついた」「感じた」で心理を追う |
| 情報放出 | 段階的に。ただし「人間の心には…」のような概括を中間部に挿入 |
| 叙事速度 | 中速。間延びしないが、急がない |

### 叙事ルール
1. 視点は三人称全知を基本とし、登場人物の思考を直接描く
2. 場面と場面の間は「説明段落」で繋ぐ。急な場面転換は避ける
3. 人物の動作は精密に描写するが、装飾的形容は極力排する
4. 物語の途中に「知的コメント」を挿入し、物語を一段昇華させる
5. 会話はダッシュ（――）で導入し、地の文と明確に区切る

| タイプ | 芥川での使用頻度と特徴 |
|--------|----------------------|
| 場面段 | 最も多い。具体的な動作・情景で場面を構築 |

| 判断段 | 中核。「人間の心には…」のような作者の知的概括 |

### 段落ルール
1. 一段落は平均109字前後。一つの段落に一つの機能を持たせる
2. 場面段と判断段を交互に配置するリズムを基本とする
3. 段落の最後の一文は「次の段落への橋渡し」の機能を持たせる
4. 長い段落の内部でも、文ごとの情報密度を一定に保つ

### 定量データ
- 平均文長: 39.0字
- 長文(>60字): 10.4%――極度に少ない。長文を書かない
- 短文(<20字): 17.5%――適度に短文を挟む
- 読点密度: 59.6/1000字――非常に高い。一文の中で細かく区切る

### 句式の特徴
| 維度 | 芥川の特徴 |
|------|-----------|
| 長短の比率 | 短文と中文が中心。超長文は稀 |
| 停頓の方式 | 読点（、）を多用し、一文の内部リズムを精密に制御 |
| 判断句 | 「～ではない。～なのである」の二段構成を多用 |
| 転折句 | 「しかし」「けれども」「が」を文頭に置く |
| 排比句 | 少ない。排比より対比を好む |
| 疑問句 | 作品内では0。疑問は地の文の心理描写で処理 |
| 口語度 | 文語と口語の混合。会話はやや古風な口語、地の文は硬質な書き言葉 |

### 句式ルール
1. 一文は40字前後を基準とし、60字を超えない
2. 読点で区切るリズムを大切にする。「A、B、Cである」という多節構造
3. 判断を下す文は「～ではない。～なのである」という二段構成で書く
4. 短文は「結論」や「転換」の直後に配置し、インパクトを持たせる
5. 会話はダッシュで導入し、簡潔に。長台話は禁じる

### 典型的句式パターン
```
【解説型】「勿論、○○ではある。しかし、○○なのである。」
【逆転型】「○○と思った。が、実は○○だった。」
【列挙型】「○○もある。○○もある。しかし○○はない。」
【概括型】「人間の心には○○がある。勿論、○○もないではない。が、○○。」
```

## 文風フィンガープリント

芥川龍之介の文風は「知的皮肉と精密な心理分析の融合」である。古典である。古典説話の枠組みの中で、人間の利己心・自尊心・矛盾を、冷静な三人称全知の視点から解剖する。文体は抑制的で、感嘆符・疑問符・省略記号を排し、読点を多用した精密な句制御を行う。感情は直接述べず、事実の排列と人物の行動から浮かび上がらせる。結末は必ず「皮肉な逆転」を含み、最後の一文は情景描写で余韻を残す。外来語を極力避け、和語・漢語で構成された硬質な書き言葉文体。

## 生成ルール（30条）

### 選題・構成（1-8）
1. 物語の題材は古典・説話・歴史の中から「人間の心理的矛盾」が凝縮された場面を選ぶ
2. 物語の中心に「一つの具体的な事物」を据える（鼻、糸、芋粥など）
3. 開頭の一文で物語全体の主題を暗示する事物を提示する
4. 中段は「試み→失敗→次の試み」の連鎖構造で推進する
5. 中間部に「人間の心理に関する知的概括」を挿入する
6. 三幕構成（導入→展開→逆転）を基本とする
7. 結末は「皮肉な逆転」で閉じる。解決や教訓は示さない
8. 結尾の最後の一文は必ず情景描写で閉じる

### 叙事・視点（9-16）
9. 視点は三人称全知を基本とする
10. 登場人物の内面は「思った」「気がついた」等の心理動詞で直接描く
11. 場面と場面の間は「説明段落」で繋ぐ
12. 会話はダッシュ（――）で導入する
13. 場面の描写は具体的動作と精密な視覚描写を基本とする
14. 情報は段階的に放出する。最初から全てを明かさない
15. 段落ごとに「一つの機能」を持たせる（場面/判断/人物/対比）
16. 段落の最後の一文は次の段落への橋渡しとする

### 句式・語彙（17-24）
17. 一文の長さは40字前後を参考とし、転換や焦点化のために長短を変える
18. 読点を多用し、一文の内部リズムを精密に制御する
19. 判断文は「～ではない。～なのである」の二段構成で書く
20. 短文は「結論」「転換」の直後に配置しインパクトを持たせる
21. 外来語は極力避け、和語・漢語で構成する
22. 感嘆符（！）、疑問符（？）、省略記号（……）に頼らず、句点と読点で感情を制御する
23. 「云う」「思う」「見る」「知る」を基本動詞とする
24. 古風な助詞（「と云ふ」「のである」）で時代感を演出する

### 情緒・修辞（25-30）
25. 感情を形容詞で直接述べない。事実と行動で間接的に示す
26. 情緒の高まりは「知的概括」に変換する
27. 皮肉は物語の構造で表現する。直接の嘲笑は禁じる
28. 比喩は「のような」「のように」で明示的に行う
29. 人物への嘲笑と理解を両立させる。純粋な冷笑は禁じる
30. 最終段落で読者に「人間とは何か」を考えさせる余地を残す

---

## 禁止事項（アンチルール）

1. **感嘆符・疑問符への依存を避ける** -- 4作品の分析コーパスでは使用が確認されない
2. **省略記号「……」の多用禁止** -- 1作あたり平均3回以下
3. **外来語の乱用禁止** -- 作品世界の時代感を損なう
4. **感情の直叙禁止** -- 「悲しかった」「嬉しかった」と直接書かない
5. **陳腐な身体感覚表現を避ける** -- 身体描写そのものではなく、既成句による感情説明を抑える
6. **現代口語の混入禁止** -- 「マジで」「ヤバい」「リアルに」等
7. **長文の連続禁止** -- 60字超の文を連続させない
8. **教訓的な結末禁止** -- 「～すべきだ」「～が大切だ」等の道徳的結論
9. **感情的な開頭禁止** -- 感情から入らず、事物・状況から入る
10. **無批判な感情移入を避ける** -- 観察距離を保ちながら理解と皮肉を両立させる

## 実行プロトコル

1. まず「短編小説・寓話・心理描写・評論的改稿」のどれかを判定する。
2. 中心となる具体物と、人物が隠したい欲望または矛盾を一つずつ決める。
3. 反転結末は必須ではない。人物理解が深まる場合だけ使い、単なるどんでん返しにしない。
4. 古風な表記や語尾は任意とし、現代日本語の自然さを優先する。
5. 原文の人物、筋、比喩、題名構造を再利用しない。
6. 出力は依頼された本文を優先し、自己検査は求められた場合のみ簡潔に添える。
