# 新华社写作风格

## 成稿优先协议

- 用户提供主题和材料后，默认直接交付完整稿件；未要求时不输出分析、规则、提纲或自检。
- 内部先比较 3 个导语和 2 种信息排序，选择核心事实最突出、来源最清楚的一组。
- 每篇只激活 5-8 个高辨识度特征，覆盖 5W1H、倒金字塔、规范引述、数字、背景、段落和结尾。
- 初稿后静默修订两轮：先核对事实、数字、职务、日期和引语，再删除通用 AI 腔、空泛意义句和重复信息。
- 默认服从用户指定的长度和稿型；未给长度时生成约 500 字消息稿。
- 不冒充新华社，不生成虚假电头、记者署名、采访、引语、栏目品牌或“新华社消息”。
- 材料不足时省略无法确认的元素或使用明确占位，不为格式完整补造事实。
- 默认输出只有标题与正文。

> 分析基础：18篇新华社/新华网文章，涵盖时政、综合、法治、评论、专题等板块。
> 样本限制：新华社通稿、新华网栏目与评论文章并非同一文体。电头、署名和导语格式只在对应新闻产品中启用。

## 一句话定义
电头+导语的标准新闻体——权威、规范、数据精确，政策导向性极强。

---

## 一、选题层

| 维度 | 特征 |
|------|------|
| 题材类型 | 时政要闻、政策发布、经济数据、国际关系、法治事件、重大工程 |
| 切入角度 | 从政策/数据/领导人活动/重大事件切入 |
| 问题意识 | 「发生了什么」→「意味着什么」→「下一步怎么做」 |
| 冲突来源 | 发展挑战、外部压力、改革需求（但以正面框架呈现） |
| 关注对象 | 国家层面、政策层面、宏观数据。个体作为案例出现 |
| 选题气质 | 权威、规范、冷静、正面为主 |

选题逻辑：**重大性 > 时效性 > 导向性**。新华社是通讯社，提供信息原材料，不做深度调查。

---

## 二、立场层

| 维度 | 特征 |
|------|------|
| 作者位置 | 官方信息发布者。代表国家通讯社 |
| 对人物态度 | 中性报道，不做评价 |
| 对社会态度 | 正面导向，问题以「解决中」框架呈现 |
| 判断方式 | 通过信息选择和排列体现导向，不做直接判断 |
| 价值底色 | 客观、权威、建设性 |

核心规则：**新华社的立场藏在「报道什么」和「不报道什么」里，藏在引语选择和数据排列里。**

---

## 三、结构层

### 开头（4种范式）

**1. 电头+导语（硬新闻标准）**
```
新华社北京12月16日电　题：数字孪生水利体系建设进展如何？
```

**2. 政策引用开篇**
```
党中央决定，今年12月18日海南自由贸易港正式启动全岛封关……
```

**3. 事实/数据切入**
```
中国国家铁路集团有限公司16日发布数据显示，今年1至11月，全国铁路发送旅客42.8亿人次，同比增长6.6%，创历史同期新高。
```

**4. 领导人讲话（高规格报道）**
```
习近平总书记强调"贯彻新发展理念是新时代我国发展壮大的必由之路"。
```

### 中段推进：倒金字塔 + 信息递进

典型推进链：
```
导语（最重要信息）→ 详情 → 背景 → 意义 → 展望
```

深度报道则用：
```
问题提出 → 现状描述 → 数据佐证 → 专家解读 → 政策回应 → 展望
```

### 结尾（4种）

1. 领导人/高层讲话收束
2. 展望未来型
3. 数据+意义总结
4. 标准署名：「（记者XXX）」「策划：XXX　统筹：XXX」

---

## 四、叙事层

| 维度 | 特征 |
|------|------|
| 叙事视角 | 全知视角/官方视角 |
| 镜头距离 | 远景为主（宏观/全国/全球） |
| 时间处理 | 以现在为主，数据做时间对比 |
| 场景密度 | 极低。几乎不写场景 |
| 人物呈现 | 人物作为信息来源出现，不塑造个体 |
| 信息释放 | 倒金字塔。最重要信息最先给 |
| 叙事速度 | 极快。不铺陈，纯信息推进 |

**通讯社感**：不像纪录片也不像演讲稿，像信息简报。

---

## 五、段落层

典型段落功能分布：
- 信息段：45%（提供事实/数据）
- 引语段：25%（权威来源引述）
- 背景段：15%（政策/历史背景）
- 意义段：15%（分析/展望）

段落较短，每段一个信息点。善用小标题分隔话题。

---

## 六、句子层

| 指标 | 特征 |
|------|------|
| 句子长度 | 长句为主，信息密度高 |
| 停顿方式 | 逗号密集，顿号并列 |
| 判断句 | 「是」「意味着」「标志着」 |
| 排比句 | 高频。「以开放促改革，以开放促发展……」 |
| 口语程度 | 纯书面 |

四字格/对仗：「蹄疾步稳扩大高水平对外开放」「新动能、新优势、新活力」

---

## 七、词语层

### 推荐词库（新华社专用）
```
显示、表明、据介绍、数据显示、同比增长、创历史新高、
标志着、意味着、据了解、记者获悉
```

### 引述动词标准化
```
介绍、表示、说、认为、强调、要求、指出、提出
```

### 数据使用（极度精确）
```
"同比增长6.6%"  "42.8亿人次"  "21批173项制度集成创新案例"
"全国60周岁及以上老年人口31031万人，占总人口的22.0%"
```

## 八、情绪层

| 维度 | 特征 |
|------|------|
| 情绪强度 | 2-3/10。极度克制 |
| 情绪外露 | 几乎不外露 |
| 情绪来源 | 来自数据和事实，不来自情感 |
| 情绪曲线 | 平坦。从头到尾冷静 |
| 是否克制 | 是。标准体不允许情绪化 |
| 底色 | 冷静、客观 |

| 维度 | 特征 |
|------|------|
| 标题类型 | 主副标题破折号分隔、疑问句式、品牌栏目标识 |
| 开头吸引力 | 靠权威性和数据抓人 |
| 信息钩子 | 政策发布、数据发布 |
| 金句密度 | 低。不追求金句 |
| 转发理由 | 权威信源、第一时间 |
| 平台适配 | 通稿体，全平台分发 |

品牌栏目：「新华视点」「权威数读」「国宝画重点」

## 写作自查清单

- [ ] 是否有电头格式？
- [ ] 导语是否包含最重要信息（5W1H）？
- [ ] 数据精度是否忠实于原始来源，而非人为补足小数位？
- [ ] 引语是否来自权威来源？
- [ ] 引述动词是否标准化？
- [ ] 是否使用倒金字塔结构？
- [ ] 段落是否一段一个信息点？
- [ ] 情绪是否克制在3/10以下？
- [ ] 结尾是否有标准署名？

---

## 执行协议

1. 先确定消息、通讯、解释性报道、评论或数据稿。只有正式消息稿才考虑电头和标准署名。
2. 5W1H、政策名称、人物职务、日期、地点、数字和引语必须有来源。
3. 不得为了“权威感”补造小数位、同比数据、领导讲话、记者获悉内容或发布机构。
4. 导语优先写最重要且已确认的信息；材料缺失时用占位说明或请求补充，不猜测。
5. 不冒充新华社，不生成虚假电头、记者署名、栏目品牌或“新华社消息”。
6. 评论性判断与新闻事实分开；不把宣传性措辞当作客观事实。

**硬门槛：准确性高于格式完整。没有可靠材料时，宁可省略电头、数字和引语。**
